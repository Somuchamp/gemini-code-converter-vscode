# Code Converter (Powered by Gemini)

A powerful Visual Studio Code extension to translate code snippets from one programming language to another using Google's Gemini AI.

## Features

*   Convert code between C++, Java, Python, Ruby, and JavaScript.
*   Clean, integrated UI inside the VS Code editor.
*   Securely stores your API key using VS Code's Secret Storage.

## How to Use

1.  Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) to open the Command Palette.
2.  Run the command `Start Code Converter`.
3.  The first time you run it, you will be prompted to enter your Google Gemini API Key.
4.  Paste your code, select the source and target languages, and click "Convert"!

## Requirements

You must have a valid Google Gemini API Key. You can get one for free from [Google AI Studio](https://aistudio.google.com/app/apikey).
